# 📊 Dashboard Comercial – Vendas e Estoque

## 📌 Sobre o Projeto
Este projeto consiste em um dashboard interativo para acompanhamento de vendas e controle de estoque, integrado a uma planilha em Excel com automações em VBA para entrada e gestão dos dados.

O objetivo é fornecer uma visão clara e rápida da operação comercial, apoiando a tomada de decisão com base em dados.

---

## 🚀 Funcionalidades

- 📈 Acompanhamento de vendas (faturamento, volume, desempenho)
- 📦 Controle de estoque atualizado
- 🔄 Lançamento de vendas automatizado via VBA
- 📊 Indicadores de performance (KPIs)
- 📅 Análise temporal (por dia, mês, período)
- 🧩 Integração entre Excel e dashboard

---

## 🛠️ Tecnologias Utilizadas

- Excel
- VBA (Visual Basic for Applications)
- Power BI / Tableau *(ajuste conforme utilizado)*

## ⚙️ Como Utilizar

### 1. Clonar o repositório
git clone https://github.com/GrassoRafael/Dashboard-Comercial.git


### 2. Abrir a base Excel
- Arquivo localizado em: `/dados`
- Habilitar macros ao abrir

### 3. Utilizar as automações
- Realizar lançamentos via formulário VBA
- Atualizar dados conforme necessário

### 4. Abrir o Dashboard
- Arquivo em `/dashboard`
- Atualizar conexões de dados (se necessário)
